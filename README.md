# 🖥️ Landing Page Restore

Projeto de **Landing Page** desenvolvido com **HTML, CSS e JavaScript**, criado para praticar construção de interfaces modernas e responsivas para apresentação de serviços ou produtos.

A página apresenta um layout organizado, com seções visuais e interações básicas para melhorar a experiência do usuário.

---

## 🚀 Tecnologias utilizadas

* **HTML5**
* **CSS3**
* **JavaScript**

---

## ⚙️ Funcionalidades

* 📄 Estrutura de **landing page moderna**
* 🎨 Estilização com **CSS**
* ⚡ Interações básicas com **JavaScript**
* 📱 Estrutura preparada para adaptação em diferentes telas
* 🧩 Código organizado e fácil de entender

---

## 📂 Estrutura do projeto

```
landingpage-restore
│
├── index.html
├── styles.css
└── script.js
```

---

## ▶️ Como executar o projeto

1. Clone o repositório

```
git clone https://github.com/seu-usuario/landingpage-restore.git
```

2. Acesse a pasta do projeto

```
cd landingpage-restore
```

3. Abra o arquivo

```
index.html
```

no seu navegador.
